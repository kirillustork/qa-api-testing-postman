# API тестирование — DemoShopping

## О проекте

Учебный проект по тестированию REST API интернет-магазина с использованием Postman.

Цель проекта — продемонстрировать навыки тестирования API, проверки бизнес-логики, валидации данных и выявления несоответствий между документацией (Swagger) и фактической реализацией API.

## 🔍 Область тестирования

В рамках проекта протестированы следующие области API:

### Products
- Получение списка товаров  
- Получение товара по ID

- Обновление товара (PUT / PATCH)

- Изменение количества товара  
- Удаление товара из корзины  

### Авторизация
- Логин пользователя  
- Работа с токеном

## Как запустить

1. Импортировать коллекцию в Postman:  
   `postman/DemoShopping.postman_collection.json`

2. Настроить переменные окружения:

- `url`  
- `login` / `password`  
- `login2` / `password2`  

Остальные переменные (`product_id`, `cart_item_id`, `key`, `random_id` и др.) формируются автоматически в процессе выполнения тестов.

3. Запускать запросы вручную или через Collection Runner

## Выявленные проблемы (Findings)

В процессе тестирования были обнаружены следующие дефекты и несоответствия:

### 1. Несоответствие типов данных (price)

- В Swagger поле `price` представлено числовым значением (например, `0`)  
- В API возвращается строка (например, `"50.00"`)  

Влияние:
- Падают проверки типов  
- Некорректная работа числовых проверок  

---

### 2. Несоответствие схемы: `id` vs `product_id`

- В Swagger используется поле `id`  
- В API используется `product_id`  

Влияние:
- Несоответствие контракту API  
- Необходима дополнительная логика в тестах  

---

### 3. Ошибка при PUT-запросе (500)

- Swagger предполагает передачу `id` в теле запроса  
- API использует `product_id` из URL  
- Конфликт данных приводит к ошибке сервера  

Влияние:
- Нестабильная работа метода обновления  

---

### 4. Несоответствие типа поля `freeShipping`

- В Swagger: boolean (`true/false`)  
- В API: числовое значение (`1/0`)  

Влияние:
- Ошибки при проверке фильтрации  
- Неконсистентность типов данных
